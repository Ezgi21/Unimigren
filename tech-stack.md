# UniMigren — Teknoloji Yığını (Tech Stack)

## 1. Önerilen Teknoloji Yığını

| Katman | Teknoloji |
|---|---|
| Frontend (Arayüz) | HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla JS) |
| AI Entegrasyonu | Gemini 1.5 Pro API (Google AI Studio) |
| Editör & Geliştirme | Cursor AI (Agent Modu aktif) |
| Yayınlama (Deploy) | Lovable veya Netlify |

---

## 2. Neden Bu Teknolojileri Seçiyoruz?

- **Düşük Öğrenme Eğrisi:** HTML/JS yapısı, karmaşık çerçeveler (React/Vue) öğrenmeden doğrudan "Vibe Coding" yapmana olanak tanır.
- **Hız ve Uyumluluk:** Cursor, düz HTML ve JavaScript dosyaları üzerinde hata yapma payı en düşük şekilde çalışır.
- **Ücretsiz ve Güçlü AI:** Google AI Studio API anahtarı tamamen ücretsizdir ve Gemini modeli karmaşık migren verilerini analiz etmekte çok başarılıdır.
- **Anında Yayınlama:** Lovable kullanarak GitHub deponu bağlayabilir ve tek tıkla uygulamanı dünyaya açabilirsin.

---

## 3. Kurulum Adımları

1. **Klasör Hazırlığı:** Bilgisayarında `unimigren` adlı bir klasör oluştur ve Cursor ile bu klasörü aç.

2. **API Hazırlığı:** [aistudio.google.com](https://aistudio.google.com) adresinden aldığın API anahtarını Cursor ayarlarındaki "Models" kısmına yapıştır.

3. **Dosyaların Oluşturulması:** Cursor'a `prd.md`, `tasks.md` ve `userflow.md` dosyalarını okutarak temel dosyaları (`index.html`, `style.css`, `script.js`) oluşturmasını söyle.

4. **Hava Durumu Entegrasyonu:** AI'dan ücretsiz bir hava durumu API'si (Open-Meteo gibi) kullanarak canlı veri çekmesini iste.

5. **GitHub Bağlantısı:** Kodun çalışan bir versiyonuna ulaştığında GitHub'da yeni bir repo aç ve dosyalarını oraya yükle.
